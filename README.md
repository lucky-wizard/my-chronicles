# Мои хроники · трекер активностей

Личный трекер физических активностей, прокачивающий персонажа. Оформление в стиле **Bioshock Infinite** (Колумбия).

- **`index.html`** — версия для компьютера
- **`mobile.html`** — версия для телефона

После публикации на GitHub Pages:

- ПК: `https://<ваш-логин>.github.io/<имя-репозитория>/`
- Телефон: `https://<ваш-логин>.github.io/<имя-репозитория>/mobile.html`

Все данные хранятся локально в браузере (localStorage) — у каждого устройства своя статистика.
