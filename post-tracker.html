Синхронизация через GitHub Gist
Данные хранятся в приватном Gist — синхронизация между телефоном и компьютером работает автоматически.
Создай на github.com → Settings → Personal access tokens. Права: только gist. Хранится в браузере, никуда не отправляется.
Найди в URL: gist.github.com/имя_пользователя/ID
